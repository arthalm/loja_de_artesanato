# loja_de_artesanato

Projeto de Linguagem de Programação 1 (2026.1). Um sistema de e-commerce de artesanatofeito feito em C++, com cadastro de clientes e artesãos, produtos (pinturas, esculturas e artesanatos) e pedidos, com salvamento dos dados em arquivo.

## Participantes

- Arthur Victor Vieira Almeida
- Leonardo Alencar de Aquino
- Jordson Albino Galvao Tavares

## Como rodar

```bash
cd src
make
./soul_arte
```

Para limpar os arquivos gerados pela compilação:

```bash
make clean
```
